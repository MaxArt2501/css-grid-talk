Ottimo! E ora?

Dobbiamo definire come usare la griglia

grid-column-start
grid-column-end
grid-row-start
grid-row-end

Possono assumere valori numerici:

.my-block {
    grid-column-start: 1;
    grid-column-end: 3;
    grid-row-start: 1;
    grid-row-end: 2;
}.my-block {
    grid-column-start: 1;
    grid-column-end: span 2;
    grid-row-start: 1;
    grid-row-end: span 1;
}

I valori possono essere negativi (partono dal fondo)

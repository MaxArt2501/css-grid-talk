Ottimo! E ora?

Dobbiamo definire come usare la griglia

grid-row-start
grid-row-end grid-column-start
grid-column-end

Possono assumere valori numerici:

.my-block {
    grid-row-start: 1;
    grid-column-start: 1;
    grid-row-end: 2;
    grid-column-end: 3;
}.my-block {
    grid-row: 1 / span 1;
    grid-column: 1 / span 2;
}.my-block
  { grid-area: 1 / 1 / 2 / 3; }

I valori possono essere negativi (partono dal fondo)

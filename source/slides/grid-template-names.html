Dalle aree alle guide

Le aree con nome generano in automatico dei nomi per le guide:

.my-grid {
    grid-template-areas:
        "sidebar main";
}
.my-block {
    grid-column: sidebar-start / main-end;
    grid-row: main-start / sidebar-end;
}
